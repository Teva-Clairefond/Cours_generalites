Qu'est-ce qu'un framework ?

Cours : https://www.youtube.com/watch?v=Ka77djMkSwg


Un frameworks est associé à un language. C'est un cadre qui impose une structure de code à l'utilisateur et qui effectue la gestion de ce code, à la manière dont le Noyau orchestre les processus.
Ce faisant, le frameworks permet aussi la simplification du code, et une tâche laborieuse en JavaScript peut devenir nettement plus aisée avec ReactJS.

Exemple :
