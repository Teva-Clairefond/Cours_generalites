# Model Context Protocol (MCP)


## Explications :

Le protocol MCP est un standard pour relier les assistants IA (LLM) à une source de donnée (Retrival) pour qu'ils puissent l'ajouter à leur contexte de raisonnement.

Architecture simple : Un MCP client qui se connecte à un MCP serveur.
