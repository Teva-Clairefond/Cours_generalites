# VScode commandes :


## Navigation :
Aller à une ligne : `Ctrl+G`<br>
Aller à un symbole (fonction, classe…) : `Ctrl+Shift+O`<br>
Aller à une définition : `F12`<br>
Revenir en arrière / avancer : `Alt+←` / `Alt+→` / `Ctrl+-` / `Ctrl+Shift+-`<br>
Recherche globale : `Ctrl+Shift+F`<br>

## Édition (gagne-temps énorme) :
Multi-cursor (ajouter un curseur) : `Alt+Click`<br>
Ajouter le prochain match à la sélection : `Ctrl+D`<br>
Sélectionner toutes les occurrences : `Ctrl+Shift+L`<br>
Déplacer une ligne haut/bas : `Alt+↑` / `Alt+↓`<br>
Dupliquer une ligne : `Shift+Alt+↓`<br>
Supprimer une ligne : `Ctrl+Shift+K`<br>
Commenter/décommenter ligne : `Ctrl+/`<br>
Commentaire bloc : `Shift+Alt+A`<br>
Indentation : `Tab` / `Shift+Tab`<br>

## Rechercher & remplacer dans le fichier :
Rechercher : `Ctrl+F`<br>
Remplacer : `Ctrl+H`<br>
Match suivant / précédent : `Enter` / `Shift+Enter`<br>
Sélectionner toutes les occurrences de la recherche : `Alt+Enter`<br>

## Mise en forme et “code action” :
Formatter le document : `Shift+Alt+F`<br>
Code actions (fix, imports, refactor) : `Ctrl+.`<br>
Renommer un symbole : `F2`<br>
Refactor (quand dispo) : via `Ctrl+.`<br>

## Fenêtres, onglets, disposition :
Afficher/masquer la barre latérale : `Ctrl+B`<br>
Ouvrir le panneau latéral de droite : `Ctrl + Alt + B`<br>
Ouvrir la palette : `Ctrl + Shift + P`<br>
Explorer / Recherche / Git / Extensions : `Ctrl+Shift+E` / `Ctrl+Shift+F` / `Ctrl+Shift+G` / `Ctrl+Shift+X`<br>
Split editor (2 colonnes) : `Ctrl+\`<br>
Basculer entre groupes d’éditeur : `Ctrl+1` / `Ctrl+2` / `Ctrl+3`<br>
Fermer l’onglet : `Ctrl+W`<br>
Réouvrir onglet fermé : `Ctrl+Shift+T`<br>
Afficher/masquer le panneau du bas (où se trouve le terminal) : `Ctrl + J`
Acceder à Codex : `Ctrl + Shift + Q`

## Git (très pratique) :
Vue Source Control : `Ctrl+Shift+G`<br>
Commit / Push / Pull / Stage… : via la palette (`Ctrl+Shift+P`) en tapant `Git: ...`<br>
Voir le diff : clic sur un fichier modifié dans la vue Git<br>

## Débogage :
Lancer débogage : `F5`<br>
Step over / into / out : `F10` / `F11` / `Shift+F11`<br>
Stop : `Shift+F5`<br>
Toggle breakpoint : `F9`<br>

## Petites commandes “confort” :
Toggle Word Wrap (retour à la ligne) : palette -> `Toggle Word Wrap`<br>
Zen Mode : palette -> `Zen Mode`<br>
Toggle Minimap : palette -> `Toggle Minimap`<br>
Changer de thème : palette -> `Color Theme`<br>

Astuce bonus (pour devenir très rapide)<br>
Dans `Ctrl+P`, vous pouvez faire :<br>
`:` puis un numéro -> aller à une ligne<br>
`@` -> chercher un symbole dans le fichier<br>
`#` -> chercher dans le fichier<br>
`>` -> Chercher une commande<br>
